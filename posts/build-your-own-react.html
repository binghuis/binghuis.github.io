构建你自己的 React

2023-05-23
从 0 到 1 实现一个简单的 React

我们将逐步从头开始重写 React，按照真正的 React 代码的架构进行步骤性实现，但不包括所有的优化和非必要功能。

如果你读过我之前的 “从头构建你自己的 React” 系列文章，不同之处在于本文基于 React 16.8，因此我们可以使用 Hooks 并且删除与类相关的所有代码。

你可以在 Didact 代码仓库中找到旧博客文章和代码的历史记录。还有一个关于相同内容的 演讲。但这篇文章是一个独立的内容。

从零开始，以下是我们将逐步添加到我们的 React 版本中的所有内容：

  • Step 1：createElement 函数
  • Step 2：render 函数
  • Step 3：并发模式
  • Step 4：Fibers
  • Step 5：渲染和提交阶段
  • Step 6：协调过程
  • Step 7：函数组件
  • Step 8：Hooks
